# grid-based-inventory
An Unity inventory system that u can store items which has volume on grid and manage its positions.


Used;
- My Own Grid System
- Scriptable Object for Storing Item Data
- Basic Input Manager to Get Player Mouse Position (wth Unity New Input System)
